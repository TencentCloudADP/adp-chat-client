# ADP Chat Component

ADP Chat Component 是一套完整的 AI 聊天组件库，基于 Vue 3 + TDesign 构建，提供了丰富的聊天界面组件。

## 特性

- **完整的聊天界面** - 提供 ChatLayout、Chat、Sender 等核心组件
- **灵活的布局** - 支持侧边栏、主内容区等多种布局方式
- **主题定制** - 支持亮色/暗色主题切换
- **国际化** - 内置中英文支持，可自定义文本
- **动态挂载** - 支持 Vue 组件和动态挂载两种使用方式
- **API 模式** - 支持配置 API 自动加载数据

## 安装

```bash
npm install adp-chat-component
# 或
pnpm add adp-chat-component
```

## 快速开始

```vue
<template>
  <ChatLayout
    :applications="applications"
    :conversations="conversations"
    :chat-list="chatList"
    @send="handleSend"
  />
</template>

<script setup lang="ts">
import { ChatLayout } from 'adp-chat-component'
import 'adp-chat-component/dist/style.css'
</script>
```
