# CustomizedIcon

SVG 雪碧图图标组件。

## 基础用法

```vue
<template>
  <CustomizedIcon name="chat" size="m" />
</template>
```

## API

<PropsTable data={[
  { name: 'name', type: 'string', required: true, description: '图标名称' },
  { name: 'size', type: "'xs' | 's' | 'm' | 'l' | 'xl'", default: "'m'", description: '图标尺寸' },
  { name: 'nativeIcon', type: 'boolean', default: 'false', description: '是否使用原生图标样式（不应用主题滤镜）' },
