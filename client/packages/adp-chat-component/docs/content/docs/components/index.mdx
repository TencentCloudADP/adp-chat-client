# 组件总览

ADP Chat Component 提供了丰富的组件，分为以下几类：

## 布局组件

| 组件 | 说明 |
|------|------|
| [ChatLayout](/docs/components/layout/chat-layout) | 完整聊天布局，包含侧边栏和主内容区 |
| [MainLayout](/docs/components/layout/main-layout) | 主内容区布局 |
| [SideLayout](/docs/components/layout/side-layout) | 侧边栏布局 |

## 聊天组件

| 组件 | 说明 |
|------|------|
| [Chat](/docs/components/chat/chat) | 聊天主组件，包含消息列表和发送器 |
| [ChatItem](/docs/components/chat/chat-item) | 单条聊天消息 |
| [ChatSender](/docs/components/chat/sender) | 消息发送器 |
| [ChatAppType](/docs/components/chat/app-type) | 应用欢迎页 |
| [BackToBottom](/docs/components/chat/back-to-bottom) | 回到底部按钮 |

## 基础组件

| 组件 | 说明 |
|------|------|
| [Button](/docs/components/basic/button) | 按钮组件 |
| [AIWarning](/docs/components/basic/ai-warning) | AI 警告提示 |
| [ApplicationList](/docs/components/basic/application-list) | 应用列表 |
| [HistoryList](/docs/components/basic/history-list) | 历史会话列表 |
| [Settings](/docs/components/basic/settings) | 设置组件 |
| [Search](/docs/components/basic/search) | 搜索组件 |
| [PersonalAccount](/docs/components/basic/personal-account) | 个人账户 |
| [LogoArea](/docs/components/basic/logo-area) | Logo 区域 |
| [CreateConversation](/docs/components/basic/create-conversation) | 新建对话按钮 |
| [CustomizedIcon](/docs/components/basic/customized-icon) | 自定义图标 |
| [SidebarToggle](/docs/components/basic/sidebar-toggle) | 侧边栏切换 |
