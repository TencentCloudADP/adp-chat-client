# ChatAppType

展示当前应用的欢迎语和推荐问题。

## 基础用法

```vue
<template>
  <ChatAppType
    :current-application-avatar="avatar"
    :current-application-name="name"
    :current-application-greeting="greeting"
    :current-application-opening-questions="questions"
    @select-question="handleSelectQuestion"
  />
</template>
```
