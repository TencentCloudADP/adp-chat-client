# FileList

展示已上传的文件列表。

## 基础用法

```vue
<template>
  <FileList
    :file-list="files"
    :on-delete="handleDelete"
  />
</template>

<script setup>
const files = [
  { uid: '1', url: 'https://example.com/file1.png', name: 'file1.png' },
  { uid: '2', url: 'https://example.com/file2.pdf', name: 'file2.pdf' },
]

const handleDelete = (index) => {
  files.splice(index, 1)
}
</script>
```
