# ChatSender

消息输入和发送组件，支持文本输入、文件上传、语音录入等功能。

## vue中使用

```vue
<template>
  <ChatSender
    :is-stream-load="isLoading"
    @send="handleSend"
    @stop="handleStop"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.ChatSender.mount('#containerId', {
  isStreamLoad: false,
  onSend: (value, fileList) => {},
  onStop: () => {}
})
</script>
```
