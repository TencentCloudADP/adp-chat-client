# RecordIcon

录音状态动画图标，用于显示录音进行中的状态。

## vue中使用

```vue
<template>
  <RecordIcon />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.RecordIcon.mount('#containerId', {})
</script>
```

## API

该组件无 Props，仅展示录音动画效果。
