# OptionCard

展示可点击的选项卡片列表，常用于 AI 回复中的快捷选项。

## 基础用法

```vue
<template>
  <OptionCard
    :cards="options"
    :send-message="handleSendMessage"
  />
</template>

<script setup>
const options = ['选项一', '选项二', '选项三']

const handleSendMessage = (text) => {
  console.log('发送消息:', text)
}
</script>
```
