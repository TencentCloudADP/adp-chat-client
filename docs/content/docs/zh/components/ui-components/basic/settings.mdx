# Settings

设置下拉菜单，包含主题切换、语言选择、退出登录等功能。

## vue中使用

```vue
<template>
  <Settings
    :theme="theme"
    :language-options="languageOptions"
    @toggle-theme="handleToggleTheme"
    @change-language="handleChangeLanguage"
    @logout="handleLogout"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.Settings.mount('#containerId', {
  theme: 'light',
  languageOptions: [{ key: 'zh-CN', value: '简体中文' }, { key: 'en-US', value: 'English' }],
  onToggleTheme: () => console.log('toggle theme'),
  onChangeLanguage: (key) => console.log('change language', key),
  onLogout: () => console.log('logout')
})
</script>
```
