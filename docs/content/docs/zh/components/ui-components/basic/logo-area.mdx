# LogoArea

显示 Logo 和标题。

## vue中使用

```vue
<template>
  <LogoArea
    :logo-url="logoUrl"
    title="ADP Chat"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.LogoArea.mount('#containerId', {
  logoUrl: '/logo.png',
  title: 'ADP Chat'
})
</script>
```
