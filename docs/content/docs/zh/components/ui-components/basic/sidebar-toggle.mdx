# SidebarToggle

侧边栏展开/收起切换按钮。

## vue中使用

```vue
<template>
  <SidebarToggle @toggle="handleToggle" />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.SidebarToggle.mount('#containerId', {
  onToggle: () => console.log('toggle')
})
</script>
```
