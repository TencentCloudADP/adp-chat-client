# CreateConversation

新建对话的快捷按钮。

## vue中使用

```vue
<template>
  <CreateConversation
    tooltip-text="新建对话"
    @create="handleCreate"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.CreateConversation.mount('#containerId', {
  tooltipText: '新建对话',
  onCreate: () => console.log('create')
})
</script>
```
