# Search

搜索输入框组件。

## vue中使用

```vue
<template>
  <Search
    placeholder="搜索会话"
    @search="handleSearch"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.Search.mount('#containerId', {
  placeholder: '搜索会话',
  onSearch: (value) => console.log('search', value)
})
</script>
```
