# PersonalAccount

显示用户头像和名称。

## vue中使用

```vue
<template>
  <PersonalAccount
    :avatar-url="avatarUrl"
    :name="userName"
    @click="handleClick"
  />
</template>
```

## html/react或其他框架中使用

```html
<div id="containerId"></div>
<script>
ADPChatComponent.PersonalAccount.mount('#containerId', {
  avatarUrl: '/avatar.png',
  name: '用户名',
  onClick: () => console.log('clicked')
})
</script>
```
