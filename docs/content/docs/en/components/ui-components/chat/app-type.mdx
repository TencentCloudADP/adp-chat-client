# ChatAppType

Display current application greeting and suggested questions.

## Usage in Vue

```vue
<template>
  <ChatAppType
    :current-application-avatar="avatar"
    :current-application-name="name"
    :current-application-greeting="greeting"
    :current-application-opening-questions="questions"
    @select-question="handleSelectQuestion"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.ChatAppType.mount('#containerId', {
  currentApplicationAvatar: '/avatar.png',
  currentApplicationName: 'App Name',
  currentApplicationGreeting: 'Hello, how can I help you?',
  currentApplicationOpeningQuestions: ['Question 1', 'Question 2'],
  onSelectQuestion: (question) => console.log('selected', question)
})
</script>
```
