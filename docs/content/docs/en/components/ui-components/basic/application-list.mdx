# ApplicationList

Display and switch application list.

## Usage in Vue

```vue
<template>
  <ApplicationList
    :applications="applications"
    :current-application-id="currentAppId"
    @select="handleSelect"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.ApplicationList.mount('#containerId', {
  applications: [],
  currentApplicationId: '',
  onSelect: (app) => console.log('selected', app)
})
</script>
```
