# LogoArea

Display logo and title.

## Usage in Vue

```vue
<template>
  <LogoArea
    :logo-url="logoUrl"
    title="ADP Chat"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.LogoArea.mount('#containerId', {
  logoUrl: '/logo.png',
  title: 'ADP Chat'
})
</script>
```
