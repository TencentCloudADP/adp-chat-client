# HistoryList

Display chat conversation history.

## Usage in Vue

```vue
<template>
  <HistoryList
    :conversations="conversations"
    :current-conversation-id="currentId"
    @select="handleSelect"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.HistoryList.mount('#containerId', {
  conversations: [],
  currentConversationId: '',
  onSelect: (conversation) => console.log('selected', conversation)
})
</script>
```
