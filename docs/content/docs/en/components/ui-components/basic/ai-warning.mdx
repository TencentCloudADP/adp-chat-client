# AIWarning

Display AI generated content warning tip.

## Usage in Vue

```vue
<template>
  <AIWarning text="Content generated by AI, for reference only" />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.AIWarning.mount('#containerId', {
  text: 'Content generated by AI, for reference only'
})
</script>
```
