# CustomizedIcon

SVG sprite icon component.

## Usage in Vue

```vue
<template>
  <CustomizedIcon name="chat" size="m" />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.CustomizedIcon.mount('#containerId', {
  name: 'chat',
  size: 'm'
})
</script>
```
