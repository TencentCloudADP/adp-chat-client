# Search

Search input component.

## Usage in Vue

```vue
<template>
  <Search
    placeholder="Search conversations"
    @search="handleSearch"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.Search.mount('#containerId', {
  placeholder: 'Search conversations',
  onSearch: (value) => console.log('search', value)
})
</script>
```
