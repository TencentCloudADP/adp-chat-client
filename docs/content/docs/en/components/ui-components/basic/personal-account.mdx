# PersonalAccount

Display user avatar and name.

## Usage in Vue

```vue
<template>
  <PersonalAccount
    :avatar-url="avatarUrl"
    :name="userName"
    @click="handleClick"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.PersonalAccount.mount('#containerId', {
  avatarUrl: '/avatar.png',
  name: 'Username',
  onClick: () => console.log('clicked')
})
</script>
```
