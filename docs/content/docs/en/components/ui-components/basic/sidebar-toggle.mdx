# SidebarToggle

Sidebar expand/collapse toggle button.

## Usage in Vue

```vue
<template>
  <SidebarToggle @toggle="handleToggle" />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.SidebarToggle.mount('#containerId', {
  onToggle: () => console.log('toggle')
})
</script>
```
