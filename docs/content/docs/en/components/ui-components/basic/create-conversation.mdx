# CreateConversation

Quick button to create new conversation.

## Usage in Vue

```vue
<template>
  <CreateConversation
    tooltip-text="New Chat"
    @create="handleCreate"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.CreateConversation.mount('#containerId', {
  tooltipText: 'New Chat',
  onCreate: () => console.log('create')
})
</script>
```
