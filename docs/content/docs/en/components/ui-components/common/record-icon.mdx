# RecordIcon

Recording status animation icon, used to display recording in progress state.

## Usage in Vue

```vue
<template>
  <RecordIcon />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.RecordIcon.mount('#containerId', {})
</script>
```

## API

This component has no Props, only displays recording animation effect.
