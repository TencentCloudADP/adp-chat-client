# OptionCard

Display clickable option card list, commonly used for quick options in AI replies.

## Basic Usage

```vue
<template>
  <OptionCard
    :cards="options"
    :send-message="handleSendMessage"
  />
</template>

<script setup>
const options = ['Option 1', 'Option 2', 'Option 3']

const handleSendMessage = (text) => {
  console.log('Send message:', text)
}
</script>
```
