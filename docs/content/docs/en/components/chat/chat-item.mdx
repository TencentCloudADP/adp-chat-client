# ChatItem

Single chat message display component.

## Basic Usage

```vue
<template>
  <ChatItem
    :item="message"
    :index="0"
    :loading="false"
    :is-stream-load="false"
    @copy="handleCopy"
    @rate="handleRate"
  />
</template>
```
